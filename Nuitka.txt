 - Компиляция. Если для x64, то в переменной среды "Path": пути для x64 должны быть над x32;
- \Python310\Scripts\
- \Python310\

 - Информация о версиях пакетов Pip:
# pip list
 - Просмотреть обновления:
# pip list -o
 - Обновить пакет:
# pip install <package> --upgrade
 - Обновить Pip:
# python.exe -m pip install -–upgrade pip
 - Установить пакет pip определенной версии:
# pip install <package>==6.3.1
 - Удалить пакет:
# pip uninstall <package>
